## 📝 ユーザーに話して良い概要 (Public Knowledge)

- 2024年の日本国内のフィッシング詐欺被害は**前年比20%増加**し、2025年も被害が拡大。特に「ネット証券・ネットバンキング・オンラインショッピング・暗号資産取引所」が狙われている。
- 「自分は大丈夫」と思っている一般ユーザーも含めて、**誰もが被害者になり得る状況**であり、専門知識がなくてもできる基本対策が重要。
- 本記事では、以下の**4つの必須対策**と**2つの上級対策**を紹介している：  
  1. **パスキーで簡単・安全ログイン**  
     - パスワードの代わりに**指紋や顔認証**でログインできる仕組み。  
     - 偽サイトでは使えず、フィッシングに強い。パスワードを覚える必要も減らせる。  
     - Google / Apple / Microsoft / Amazon / メルカリなど主要サービスが対応済み。
  2. **パスワード管理アプリで楽々管理**  
     - パスワードの使い回しやブラウザ保存は危険。  
     - 数字・記号を含む**長くて複雑なパスワード**が推奨される一方で、人間が覚えるのはほぼ不可能。  
     - アプリに任せて「自分で考えない・覚えない・入力しない」体制を作るのが現実解。
  3. **二要素認証（2FA）で二重ロック**  
     - パスワードに加えて、スマホに届く6桁コードなどを入力する仕組み。  
     - ID/パスワードが漏れても、2FAを突破されない限りログインされにくい。
  4. **怪しいリンクを見抜く方法**  
     - SNSやメールのリンクからのフィッシングが多い。  
     - PCではカーソルを合わせてURLを確認、スマホでは長押しでプレビュー表示し、本当のURLを必ず見る。  
     - 「amaz0n.com」「arnazon.com」「amazon-jp.info」など、似せたURLに注意。短縮URL（bit.ly等）も要警戒。
- 上級者向けの追加対策として、  
  5. **メールアドレスを使い分ける**（Apple メールを非公開、SimpleLogin、StartMail、Gmailエイリアス、NordPassのメールマスキングなど）  
  6. **VPNで通信を守る**（特にカフェや空港のフリーWi-Fi利用時）  
  が解説されている。
- **おすすめパスワード管理アプリ**は用途別に：  
  - 初心者向け・有料（家族プランあり）：**NordPass**（生体認証・パスキー対応）  
  - Apple製品ユーザー向け：**Apple パスワード（iCloudキーチェーン）**  
  - セキュリティ重視派向け：**Bitwarden**（オープンソース・世界中で監査）
- **おすすめ二要素認証アプリ**：  
  - 一般的で普及：**Google Authenticator**  
  - セキュリティ志向・透明性重視：**2FAS**（オープンソースで世界的に高評価、クラウドバックアップ機能あり）
- メールアドレスの使い分けでは：  
  - Apple「メールを非公開」：ランダムアドレスを発行し、そこからの送信も可能。  
  - SimpleLogin / StartMail：既存アドレスを隠しつつ、エイリアスを無制限に作成可能。  
  - Gmailエイリアス：`yourname+service@gmail.com` 形式でサービスごとに管理。  
  - NordPass「メールマスキング」：受信用エイリアスを作成できるが、**送信用には使えない**点に注意。
- VPNは、  
  - 公共Wi-Fi利用時や海外サイトアクセス、プライバシー重視のユーザーに有効。  
  - 無料VPNは個人情報が売られるリスクがあるため推奨されず、**信頼できる有料VPN（月額500円〜程度）**が勧められている。  
  - 日本の金融サービスはVPN経由アクセスを制限・ロックする場合があるため、特に海外からの利用では注意が必要。
- 記事末尾には、**今すぐチェックできるセキュリティ対策チェックリスト**と、  
  1) パスワード管理アプリ導入  
  2) パスキー・2FA有効化  
  3) メール使い分け＋VPN＋定期的な取引履歴確認  
  の3ステップで「今すぐ何をすべきか」がまとめられている。
